Quick Start Guide

Set up your Tenant

Configure SAP Hybris Profile Tag

Follow the instructions at Profile Tag to instrument your website(s) so that visitor activity can be collected and aggregated by the SAP Hybris Profile.

Configure Real-Time Reporting Service

Create or edit an existing YaaS project and subscribe to both of the following packages:
Profile Core Services (Beta)
Profile Services for Commerce (Beta)
Follow the complete instructions at to enable your big data to be collected / aggregated by the SAP Hybris Profile, and subsequently queried in real time via the Real-Time Reporting Service.

Subscribe to the Profile Live Dashboard YaaS package

Open the same YaaS project you created / configured in the previous step and add a subscription to the Profile Live Dashboard [BETA] v1 package

Configure Client Callback

Create or edit an existing OAuth2 Client on the same YaaS project you created / configured in a previous step and add the following callback for the Profile Live Dashboard:
https://livedashboard-v1-prod.us-east.internal.yaas.io/callback/

Configure Website Display Namek

After you subscribe to the Profile Live Dashboard package, a Builder Module named ‘Live Dashboard Configuration’ will appear in the navigation menu to the left under your project. Select that Builder Module and then open the Configuration tab, enter a display name of your choice for the brand / website which will be displayed in the Live Dashboard UI at runtime.

Launch the App

Find your tenant-specific URL

In the Builder Module named ‘Live Dashboard Configuration’ which appears in the navigation menu to the left under your project, open the Application tab. Find the name of the client from the previous step for which you configured the callback. Click the Application URL to its right. This is the tenant-specific URL which runs the Profile Live Dashboard for the website(s) you’ve tagged with SAP Hybris Profile Tag. Once clicked you will be prompted for login to YaaS or you will be logged in automatically (via SAP Cloud Identity).

Share your tenant-specific URL

You may share that same tenant-specific application URL with any user to whom you have granted project member privileges for this same project.

Customize the App

Manage Referrers

In the Builder Module named ‘Live Dashboard Configuration’ which appears in the navigation menu to the left under your project, open the Configuration tab and click the card which reads, “Add Referrers”. Enter a brand-appropriate display name and the color of your choice for the popular referrers to your site. Stars which represent a group of visitors from the same referrer will be shown in this color.

Integrate the App with SAP Systems

SAP Hybris Marketing Cloud and OnPrem 1708+

A custom integration scenario named ‘Marketing – SAP Hybris Profile Live Dashboard Integration (SAP_COM_0097)‘ allows you to enter your tenant-specific URL for the Profile Live Dashboard application as the redirect action to be taken when the user clicks on a Fiori Tile from the SAP Hybris Marketing application (both Cloud and OnPrem) beginning with the 1708 release. The Fiori Tile is located under ‘Contacts and Profiles’.

SAP Hybris Customer Journey Manager 1708++

An additional callback can be entered on your OAuth2 client enabling you to call the following endpoint from another YaaS application in the case you’d like that application to have a glimpse / preview of an abbreviated version of the Profile Live Dashboard’s Live Statistics Header. For example, the SAP Hybris Customer Journey Manager app beginning with release 1708 has added an animated Fiori Tile to which it streams the output from the following “tile-specific” application URL:
a. Tile-Specific Callback =
https://livedashboard-v1-prod.us-east.internal.yaas.io/callback_tile/

b. Tile-Specific Application URL =
https://livedashboard-v1-prod.us-east.internal.yaas.io/login_tile/[your_unique_clientId]
